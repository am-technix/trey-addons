Print Formats Delivery Label Picking

Printing poduct label from picking.

Este módulo permite la impresión de etiquetas de producto a partir de una albarán dado, permitiéndonos escoger la disposición de las etiquetas.

Utilización

  • Una vez estemos en la ficha de producto haremos click en el desplegable "Más" y a continuación seleccionaremos "Imprimir etiquetas".
  • Se abrirá un wizard dónde seleccionar tipo de etiqueta y la disposicioń.
