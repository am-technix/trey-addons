Bootstrap-compatible content blocks for drag and drop

Utilización

  • - Una vez estemos en la página del sitio web dónde queramos añadir el bloque, pulsaremos sobre el botón editar del menú Odoo y arrastraremos el bloque correspondiente del menú lateral a la zona de la página deseada.
