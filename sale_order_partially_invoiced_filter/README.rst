====================================
Sale Order Partially Invoiced Filter
====================================

Módulo para añadir el filtro de "Parcialmente facturados" a los pedidos de
venta.


Uso
===

En la vista tree de los pedidos de venta, estará disponible el filtro
"Parcialmente facturados".
