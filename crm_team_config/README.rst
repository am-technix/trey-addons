crm_team_config
===============

Este módulo añade los siguientes campos a los equipos de ventas:

    - Almacenes.

    - Diarios de pago.

    - Diarios de facturación.

Además añade reglas de seguridad para que cada usuario vea sólo los registros relacionados con su equipo de ventas.
