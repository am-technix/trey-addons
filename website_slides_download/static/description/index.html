Website Slides Download

Este módulo añade un enlace en la presentación que permite la descarga del documento pdf en función de la seguridad establecida.

Tabla de contenidos

  • Uso
  • Créditos
    • Autor

Uso

  1. Ir a "Sitio web > Presentaciones".
  2. Seleccionar una presentación.
  3. En la pestaña "Sitio web" indicar en el campo "Descargar con seguridad" el nivel de seguridad que queremos para la presentación.
