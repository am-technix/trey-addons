Sale purchase service group type

Type of grouping for the purchase of services

Cuando marcamos un producto como servicio y lo configuramos para que haga un pedido de compra por defecto, si hay una solicitud de compra para ese proveedor y tiene una linea con ese artículo, suma a la cantidad solicitada la que proviene del pedido de venta. Este módulo podemos cambiar este comportamiento agregando un campo tipo con el que controlamos como deben de hacerse estas agrupaciones.
