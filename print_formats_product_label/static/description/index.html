Print Formats Product Label

Adding wizard and small product label for producs

Este módulo permite la impresión de etiquetas de producto de forma personalizda a través del uso de un wizard.

Tabla de contenidos

  • Utilización
  • Notas

Utilización

  • Una vez estemos en la ficha de producto haremos click en el desplegable "Más" y a continuación seleccionaremos "Imprimir etiquetas".
  • Se abrirá un wizard dónde seleccionar tipo de etiqueta.

Notas

Si desea añadir otro informe al selector del wizard, el nombre debe contene la cadena '(product_label)'.
